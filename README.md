Demonstrates building and deploying a Python to use with PySpark PyConnect on AWS EMR Serverless.

1. Run `deploy_library_to_aws.ps1` in the `spark` directory, see https://spark.apache.org/docs/latest/api/python/tutorial/python_packaging.html.

2. Copy  `serverless_demo.ipynb` to the EMR Studio Workspace.  Replace the `{S3_bucket}` in the conf json with your S3 bucket.
